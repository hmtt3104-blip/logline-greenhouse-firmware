# Contributing

## Scope

Contributions should keep this repository focused on sanitized greenhouse firmware experiments.

## Rules

- Do not commit secrets or local configuration.
- Do not commit private IPs or live topology.
- Do not add logs, backups, binaries, or generated runtime outputs.
- Document hardware assumptions.
- Mark unverified behavior clearly.
- Keep experiments small enough to review.

## Before publishing changes

Run a safety scan for credentials, private IPs, local paths, logs, binaries, and deployment details.
